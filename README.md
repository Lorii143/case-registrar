# Case Registrar System

## Overview
This project is a case management system designed to help organizations track and manage cases across different departments such as litigation, medical, and psychosocial support.

## Objective
To develop a structured system that improves:
- Case tracking
- Data organization
- Reporting efficiency

## Features
- Case registration (case code, year, court, region, county)
- Case classification (type, status, closure type)
- Timeline tracking (filing date, last updated)
- Department-level usage (litigation, casework, rehab)

## Tools & Technologies
- JavaScript / HTML / CSS 
- Postgres 
- Excel / CSV 

##  Use Case
This system is designed for organizations handling sensitive case data such as:
- Human rights organizations
- Legal aid institutions
- Investigation units

## Future Improvements
- Dashboard for case analytics
- Role-based access control
- Integration with reporting tools
